Using FRS/GMRS Radios: Setup & Etiquette

Category: Preparedness & Gear • Last updated 2025-08-10

Objective

Set up simple, reliable radio comms that work when phones don’t, and use them without chaos.

Setup

  • Pick a primary channel and a CTCSS/DCS code.
  • Label radios with channel plan; program if model allows.

Etiquette

  • PTT: press, pause half‑second, speak clearly.
  • Short transmissions; avoid stepping on others.

Message Format

Who–Where–Status–Needs–Next. Example: “WILLIE—HOME—OK—WATER LOW—GOING STORE 18:00.”

Range Reality

  • Urban: 0.5–2 km typical; suburban/open: better with elevation.

Real Example

Neighborhood used FRS Ch 3 / CTCSS 18 with twice‑daily windows to share updates during an outage.

Checklist

  • Radios + spare batteries
  • Cheat card with codes
  • Headsets for noisy areas

Contingencies

  • Traffic heavy → shift to alternate channel on the plan.

After-Action

Run a weekly radio check so everyone stays proficient.
